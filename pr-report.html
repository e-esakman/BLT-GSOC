Google Summer of Code - PR Analytics

Track pull request contributions across OWASP GSoC projects

Live Analytics Dashboard

The PR Analytics dashboard requires a live backend connection to the OWASP BLT application to fetch and display real-time pull request data from GitHub.

This static GitHub Pages site cannot display live PR data, but you can access the full interactive dashboard on the main BLT application.

View Live Dashboard on BLT

Dashboard Features

Yearly Trends

Track PR contributions year-over-year across all OWASP GSoC projects with interactive charts showing growth and participation trends.

Repository Stats

View detailed statistics for each repository including total PRs, unique contributors, and contribution patterns.

Contributor Metrics

Analyze contributor activity with metrics on PR counts, merge rates, and participation across different GSoC periods.

Year Filtering

Filter data by specific GSoC years to compare performance and track improvements across different program cycles.
